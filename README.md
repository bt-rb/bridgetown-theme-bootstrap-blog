## System requirements
- Ruby >= `2.5`
- Node >= `10.13`
- Bundler
- Yarn

## Installation
Execute following commands to install:

```shell
bundle install && yarn install
```

## Getting started

Start developing by running the following command:

```shell
yarn start
```

Your site should now be live at [localhost:4000](http://localhost:4000).
